# Movie Explorer

A responsive movie discovery app built with React, React Router, and Tailwind CSS,
powered by the TMDB API.

## Features
- Browse trending & popular movies
- Full movie details: genres, budget/revenue (formatted), release date
- Search with sorting by popularity and release date
- Persisted favorites (add/remove, saved to localStorage)
- Full-bleed responsive hero banners
- Mobile-first responsive design

## Stack
React 19 · React Router · Tailwind CSS · TMDB API · Vite
